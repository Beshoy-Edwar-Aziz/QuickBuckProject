# **QuickBuck graduation project**
## **what is QuickBuck? **
QuickBuck is a graduation project in which users (Employees || Employers) are given chance to view their needs for labour and providing a chance to reflect labour market needs.
The project mainly aims to offer employees & employers a better chance of achieving success by giving employers to communicate and manage the needs of their companies and to offer fresh employees a better chance to enter the market.
## what are the technologies used to build this project?
Project is built using dotNet core 6, linq, mssql server for relational database. SignalR is also used to provide real-time chatting system between employers and employees.
